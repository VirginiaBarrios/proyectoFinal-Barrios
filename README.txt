Descripción del Proyecto
Blog realizado para usuarios que deseen subir post, con una interfaz amigable que permite tener un perfil y poder buscar post de otros usuarios.


Los usuarios pueden realizar las siguientes accciones:

Cargar posts
Visualizar post propios y de otros usuarios.
Buscar post propios y de otros usuarios.
Ademas de contar en caso de requerirlo, un superusuario administrador.

Para buscar un post se debe hacer click en el boton INICIO en la esquina superior izquierda, el usuario vera los distintos post subidos  y un boton abajo a la derecha que dice "buscar posts" al cliquearlo aparecerá un campo donde se ingresa el autor del post a buscar y un boton de búsqueda, luego apareceran todos los post de ese mismo autor.
Para ingresar a los formularios se debe hacer click a los botones los cuales se encuentran en el margen superior.
Para  crear un usuario se debe hacer click en el boton de registro donde aparecera un formulario a rellenar.
Para inciar sesion se debe hacer click en el boton de iniciar sesion en el cual aparecera un formulario para ingresar.
Para ver el perfil de usuario se debe hacer click en el boton "Bienvenido (nombre de tu usuario)" el cual llevara a una vista con los datos e imagen de perfil. Asi mismo se puede encontrar dentro de la misma el boton para agregar una foto de perfil y rellenar datos extra del perfil de usuario.
Para crear un post se debe hacer click en el boton de posteos que lleva a la vista donde se encuentra un formulario con los distintos campos requeridos y un editor de texto avanzado para realizar tu post.
